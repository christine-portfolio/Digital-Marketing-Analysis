# Digital-Marketing-Analysis
This repo contains a beginner-friendly marketing report analysis project done in Powerbi.

## Tools Used
-Power Query Editor
-Powerbi

## Project Overview
This project analyzes the return on maerketing investments and its financial impact on digital marketing.
It explores ROMI by campaigns, category and performance of the campaign using interactive dashboards.

## What I Learned
- How to clean messy data using formulas.
- How to summarize KPIs using charts and cards.
- How to visualize insights in PowerBi.
